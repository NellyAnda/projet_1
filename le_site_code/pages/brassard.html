Le brassard Zik+ Run Belt

Vous qui aimez courir en musique, le brassard Zik+ Run Belt noir vous accompagne pour vos entraînements de courses à pied. Mince et ultra-léger, il se fait oublier pour que vous restiez à l'aise et concentré sur vos performances.
